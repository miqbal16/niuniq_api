# NIUNIQ API

> Backend API for Niuniq Web Apps

## Usage
 
 Renam config "config/config.example.env" to 
 "config/config.env" and update the values/settings to your own

 ## Instal Dependencies
 
 nmp install
 
 ## Run App

 # Run in dev mode

  npm run dev

 # Run in production mode

  npm start
